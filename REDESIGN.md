## 3. Logo：豹变·叶序螺旋

- **基准分析**：SJA 的 logo 是三层含义链——苹果（公司）→ 牛顿（掉落的苹果）→ 思考（Think Different）。
  我们的对等单位需要同等的深度，单层的「萌芽/小树」不够。
- **定稿 E1（`logo-spiral.svg`）**：7 个圆点按黄金角 137.5° 叶序排列，y 向 1.18 拉伸。含义链：
  物 = 豹斑即种子 → 典故 = 君子豹变，其文蔚也（首页副标题）→ 主张 = 文即是纹，写作表达就是成纹的过程；
  形式 = 叶序是自然生长的数学秩序。
- 落点：页头、菜单面板头、favicon（同组圆点缩放）。16 个落选候选已清理，仅留正式版。（`1ba8d7b` `66bdadc`）

## 6. 视频修复（根因最值得记录）

- **症状**：封面只染顶部一小条、下面全黑；视频迟迟不可播。
- **根因**：手机直出的 MP4 把 `moov` 元数据块放在文件末尾，浏览器 `preload="metadata"` 要拖着下载找它。
- **修法**：纯 Python 做 faststart 重排——`moov` 提到 `mdat` 之前，并逐条修正 stco/co64 块偏移（+moov 长度）。
  验证：首块偏移恰好等于 mdat 数据起点。编码本身是 H.264（avc1），无需转码。（`9652d8c`）
- 修后：/vlog/ 封面满铺、时长可读；首页视频进视口自动静音播放。

## 6.5 首页卡片配图补齐（07.19）

- **家园卡片**：弃用文字占位卡，选用 4096×2160 地球地平线壁纸（四张候选中分辨率最高、构图最完整），
  居中裁 3:2（1440×960，66KB）。文案「地球归人类，AI 驶向远方」与画面直接扣题，
  深黑图成为白页上的视觉锚点。（`e8f414e`）
- **随记卡片**：用户供图「悬浮牛顿」（640×640 方图，油画风）。取 y=40–467 横带裁 3:2——
  上留空中苹果、下保人物全身，Lanczos 放大至 1440×960 并轻锐化（181KB）。
  与 logo 的苹果→牛顿→思考含义链形成呼应。（`e7fae89`）
- 至此首页五个模块（Projects / Writing / Photos / Vlog / 随记）配图全部就位，无占位图。

## 7. 经验备忘（下次少踩坑）

- favicon 改版必须带版本号参数，否则用户端长期不更新。
- 视频上线前检查 moov 位置（手机剪映/直出基本都在尾部）。
- Windows 下 Python `os.replace` 覆盖被占用文件会 PermissionError；改用 bash `rm + mv`。
- Cloudflare Pages 已删除的静态资产在边缘缓存里最长存留 4 小时，不是删错。
- Playwright 截本站：`waitUntil: 'domcontentloaded'` 并拦截 `*.mp4`，否则字体/长连接会拖超时；
  无头 Chromium 不带 H.264 解码，验证视频要用 `channel: 'chrome'`（本机真 Chrome）。
